## 実装タスク（requirement.md 対応）

### MVP（初期リリース必須）

- [ ] プラグイン `redmine_kanban` の追加（権限・メニュー・ルーティング）
- [ ] ボード画面（通常モード）
  - [ ] 列=ステータス（標準モード）で表示
  - [ ] スイムレーン：なし / 担当者（デフォルト）を切替
  - [ ] カード表示：件名 / 担当者 / 期日 / 優先度 / Blocked / 停滞日数
- [ ] DnD による列移動（ステータス更新）
  - [ ] 権限なしは DnD 無効
  - [ ] ワークフロー不許可は必ずロールバック＋理由表示
  - [ ] 失敗時は常に元位置へ戻す
- [ ] WIP 制御（必須）
  - [ ] 単位：列単位 / 列×レーン単位（設定で選択）
  - [ ] 超過時：禁止（推奨）/ 警告のみ（設定で選択）
  - [ ] 列ヘッダに「件数 / 上限」を表示し、超過を強調
- [ ] 停滞（Aging）検出（必須）
  - [ ] `updated_on` からの経過日数を表示（例 `5d`）
  - [ ] 閾値（注意/危険）を設定可能にして強調
  - [ ] Closed 列はデフォルトで除外
- [ ] Blocked 表示（必須）
  - [ ] カスタムフィールド（bool + reason）を設定で指定
  - [ ] Blocked は強調し、理由は短文表示（可能な範囲）
  - [ ] 「Blocked のみ」フィルタ
- [ ] フィルタ（必須）
  - [ ] 担当者（自分/未割当/全員/指定）
  - [ ] 件名検索
  - [ ] 期限（期限切れ/今週/未設定）
  - [ ] Blocked のみ
- [ ] タスク追加（必須）
  - [ ] 列ヘッダの「＋」から作成（初期値：プロジェクト＋列）
  - [ ] レーン×列セルの「＋」（スイムレーン有効時：推奨）
  - [ ] 最小フォーム：件名/担当者/期日/優先度/Blocked/説明
  - [ ] 成功時：即座に該当位置へ追加、失敗時：項目単位エラー
